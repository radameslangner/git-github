trackerar arquivos

git add nomedoarquivo ( adiciona arquivo a staged area )

git add -A (adiciona todos na staged area)

git add . ( adiciona a staged area só o que esta na pasta atual)

git add * ( add da pasta atua, não adicona os arquivos deletados  )


git reset nomedoarquivo ( remove da staged area )
git reset ( remove tudo da staged area )
